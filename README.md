# Project 9: Spots final

## Project Functionality

This project is made so all the elements are displayed correctly on popular screen sizes. In addition, this project also demonstrates form modals and form validation as well. Javascript was used for its implimentation. In addition, i've used the API that was provided by TripleTen for the use of the proect.
